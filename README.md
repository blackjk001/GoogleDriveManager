# GoogleDriveDownloader

Локальный WPF-менеджер для работы с Google Drive на Windows. Загружайте файлы перетаскиванием, управляйте несколькими аккаунтами, просматривайте локальный диск и облако в одном интерфейсе.

## 📋 Возможности

- ✅ Подключение нескольких Google-аккаунтов
- ✅ Просмотр файловой системы Windows и Google Drive в двухпанельном интерфейсе
- ✅ Drag&Drop скачивание файлов из Drive в локальные папки
- ✅ Менеджер загрузок с прогресс-баром и возможностью отмены
- ✅ Контекстные операции: копировать, вырезать, вставить, переименовать, удалить
- ✅ Множественное выделение (включая рамочное выделение)
- ✅ Сортировка по имени и размеру
- ✅ Современный интерфейс

## 🖥️ Требования

- **.NET Framework:** 4.7.2 или выше
- **Visual Studio:** 2022 (для сборки)
- **Google API:** Требуется настройка Google Cloud Project (см. раздел Установка)

## 🚀 Установка и запуск

### Вариант 1: Прямая сборка (для разработчиков)

1. **Клонируйте репозиторий**

2. **Откройте решение в Visual Studio**
- Файл → Открыть → Решение... → выберите `GoogleDriveDownloader.sln` из клонированной папки

3. **Восстановите NuGet пакеты**
- Visual Studio автоматически восстановит зависимости при открытии или через __Tools > NuGet Package Manager > Restore NuGet Packages__

4. **Получите `client_secrets.json` от Google**
- Перейдите в [Google Cloud Console](https://console.cloud.google.com/)
- Создайте новый проект
- Включите **Google Drive API**
- Создайте **OAuth 2.0 credentials** (тип: Desktop application)
- Скачайте JSON и переименуйте в `client_secrets.json`
- Поместите файл рядом с исполняемым файлом (`bin\Release\client_secrets.json` или `bin\Debug\client_secrets.json`)

5. **Запустите приложение**
- Нажмите **F5** в Visual Studio или запустите `.exe` из папки `bin`

### Вариант 2: Релиз (готовый exe)

1. Скачайте последний релиз из раздела **Releases**
2. Распакуйте архив
3. Поместите `client_secrets.json` в папку с `GoogleDriveDownloader.exe`
4. Запустите `GoogleDriveDownloader.exe`

## 📖 Использование

### Первый запуск
1. Нажмите кнопку **+ Добавить** справа
2. Выберите аккаунт Google в браузере
3. Авторизуйте доступ к Google Drive
4. Вы вернетесь в приложение с подключенным аккаунтом

### Навигация
- **Двойной клик** на папку → перейти внутрь
- **Кнопка 🡅** → вернуться на уровень выше
- **Левая панель** → локальный диск Windows
- **Правая панель** → Google Drive

### Скачивание файлов
1. Выделите файл(ы) в правой панели (Google Drive)
2. **Перетащите** в левую панель (локальная папка)
3. Загрузка начнется автоматически
4. Прогресс отобразится в менеджере загрузок внизу экрана

### Операции с файлами
- **Копировать** → Правый клик → Копировать (работает внутри Drive и между Windows/Drive)
- **Вырезать** → Правый клик → Вырезать
- **Вставить** → Правый клик → Вставить
- **Переименовать** → Правый клик → Переименовать
- **Удалить** → Правый клик → Удалить
